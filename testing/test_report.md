## РЕЗУЛЬТАТЫ ПО ТИПАМ ТЕСТИРОВАНИЯ

### Функциональное тестирование
**Тестирование модуля "Каталог товаров":**
Проверено 28 тест-кейсов, из них 25 успешно пройдены. 
Найденные проблемы: 
1. При фильтрации по цене от 5000 до 10000 рублей иногда показываются товары ценой 4500 рублей (дефект #FUNC-007).
2. Сортировка по рейтингу работает некорректно при одинаковых оценках (дефект #FUNC-012).
3. Быстрая прокрутка каталога на мобильных устройствах вызывает подтормаживания.

**Тестирование модуля "Корзина":**
Проверено 22 тест-кейса, из них 20 успешно пройдены.
Основные успехи: корректное добавление/удаление товаров, расчет общей суммы, применение промокодов.
Проблемы: при обновлении страницы иногда сбрасывается количество товаров (дефект #FUNC-015).

**Тестирование модуля "Оформление заказа":**
Полный пользовательский путь от корзины до подтверждения заказа проверен в 15 сценариях.
Критический дефект обнаружен на этапе оплаты: при использовании тестовой карты с истекшим сроком система не показывает адекватную ошибку, а зависает (дефект #CRIT-001, исправлен).

**Тестирование модуля "Личный кабинет":**
Проверена работа с историей заказов, настройками профиля, сменой пароля.
Обнаружен дефект безопасности: при смене email не требуется подтверждение старого адреса (дефект #SEC-003, исправлен).

### Пользовательское тестирование
**Сессии с тестовой группой:**
Привлечено 12 тестировщиков из числа одногруппников. Каждый выполнил 5 ключевых сценариев:
1. Регистрация и первый заказ
2. Поиск конкретного товара
3. Использование фильтров в каталоге
4. Оформление возврата товара
5. Настройка уведомлений

**Ключевые метрики юзабилити:**
- Среднее время выполнения задачи "Найти товар и добавить в корзину": 1 минута 45 секунд
- Количество пользователей, выполнивших все задачи без помощи: 9 из 12 (75%)
- Оценка удобства интерфейса по 5-балльной шкале: 4.2

**Основные замечания от пользователей:**
1. Кнопка "Оформить заказ" слишком мелкая на мобильной версии
2. Неочевидно, где находится история заказов в личном кабинете
3. При поиске товара не показываются подсказки
4. Процесс регистрации слишком длинный (требует много полей)

**Положительные отзывы:**
1. Дизайн современный и приятный глазу
2. Процесс оформления заказа понятный и логичный
3. Хорошая скорость загрузки страниц
4. Удобная система фильтрации товаров

### Интеграционное тестирование

## КРИТИЧЕСКИЕ НАЙДЕННЫЕ ДЕФЕКТЫ

### Дефект #CRIT-001: Зависание системы при оплате просроченной картой
**Приоритет:** Критический
**Статус:** Исправлен
**Описание:** При вводе тестовой карты с истекшим сроком действия система не показывает ошибку, а уходит в бесконечную загрузку.
**Влияние:** Пользователь не может завершить заказ, не понимает причину.
**Воспроизведение:** 100%
**Исправление:** Добавлена валидация срока действия карты на фронтенде, улучшена обработка ошибок на бэкенде.

### Дефект #CRIT-002: Потеря данных корзины при смене браузера
**Приоритет:** Критический
**Статус:** Исправлен
**Описание:** Если пользователь добавляет товары в корзину в Chrome, а затем открывает сайт в Firefox, корзина пустая.
**Влияние:** Плохой пользовательский опыт, потеря продаж.
**Воспроизведение:** 100%
**Исправление:** Реализовано сохранение корзины на сервере, а не только в localStorage.
